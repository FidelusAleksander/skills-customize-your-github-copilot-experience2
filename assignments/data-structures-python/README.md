# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Learn how to use Python's built-in data structures: lists, dictionaries, sets, and tuples. Practice creating, modifying, and accessing data in each structure through hands-on coding tasks.

## 📝 Tasks

### 🛠️ Task 1: Working with Lists

#### Description
Create a Python program that demonstrates how to create a list, add and remove elements, and access items by index.

#### Requirements
Completed program should:

- Create a list of at least 5 items
- Add a new item to the list
- Remove an item from the list
- Print the list and access an item by its index


### 🛠️ Task 2: Using Dictionaries

#### Description
Write a program that uses a dictionary to store and retrieve information about students and their grades.

#### Requirements
Completed program should:

- Create a dictionary with at least 3 student names as keys and grades as values
- Add a new student and grade
- Update an existing student's grade
- Print all students and their grades


### 🛠️ Task 3: Sets and Tuples

#### Description
Practice using sets to store unique values and tuples to store immutable data.

#### Requirements
Completed program should:

- Create a set of unique numbers and add/remove elements
- Show how duplicate values are handled in sets
- Create a tuple with at least 3 elements and demonstrate accessing its values
